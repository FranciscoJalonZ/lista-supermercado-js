#  Mi supermercado AKI

Una solución digital personalizada para la gestión y control de compras diarias. Este proyecto demuestra la integración de una interfaz de usuario moderna con lógica de persistencia de datos.

##  Funcionalidades
- **Gestión de Inventario:** Registro de productos con validación de cantidades.
- **Interactividad Total:** Marcado de productos comprados y eliminación dinámica de ítems.
- **Sincronización Local:** Implementación de `LocalStorage` para mantener la lista activa tras cerrar el navegador.
- **Panel de Control:** Visualización en tiempo real de productos totales, completados y pendientes.

##  Diseño Personalizado
La interfaz utiliza una paleta de colores exclusiva definida para este proyecto:
- **Fondo:** Dark Navy (#1a1a2e)
- **Componentes:** Light Cyan (#e0f7fa)
- **Acciones:** Vibrant Rose (#e94560)

##  Tecnologías
- HTML5 Semántico
- CSS3 (Layout con Flexbox y efectos de transición)
- JavaScript Vanilla (Gestión de arreglos y persistencia JSON)
